# Selenium POM Test Framework (Python)

This repository is a minimal Page Object Model (POM) skeleton using Selenium WebDriver in Python.

Structure:
- pages/: Page objects
- tests/: Test cases and pytest fixtures
- utils/: Utilities (driver factory, config, logging, wait helpers)

Notes:
- Uses webdriver-manager to manage browser binaries.
- Tests use pytest.

This skeleton is generated because no BDD feature files were provided. Add feature-driven tests or adapt tests/test_login.py as needed.
